# Employee Management System

## Setup EMS

##### Software you need | Application stacks
- Java 17 (OpenJDK17 `or above`)
- Maven (Apache Maven 3.9.4 `or above`)
- Cassandra
- Using H2 In-memory DB
- docker (for testing purpose)

##### Running ems 
- run runApp.sh
